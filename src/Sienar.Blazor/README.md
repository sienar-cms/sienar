# Sienar.Blazor

This package contains the basic Sienar application architecture, including a plugin that sets up the basic core infrastructure. If you want to create your own dashboard UI and rules for things like login, you should use this package. If you want Sienar to provide these utilities for you, you should use the `Sienar.Cms` package instead.

For more information, see the [Sienar docs](https://sienar.levesque.dev).
