# Sienar.Plugin.Cms.DevMode

This plugin allows developers to rapidly set up their development environment by enabling the creation of Sienar's required roles and new admin accounts without requiring a logged in user. **NOTE: THIS PLUGIN MUST NOT BE USED IN PRODUCTION.** Using this plugin in production could have catastrophic consequences for the security of your application because it allows any unauthenticated user to create an administrator account for themselves! This plugin must **only** be used on local development machines.

For more information, see the [Sienar docs](https://sienar.levesque.dev).
